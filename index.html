Small demo about "Inventory Turnover"

In this case, Inventory Turnover is measured in days. The metric shows how many days it takes to empty a warehouse.

First of all, we extract raw data from ERP. This data represents inventory state and changes between 10/2019 and 12/2019 for small part of the items. The raw data saved in format of text file with such columns:

  • Date
  • Nomenclature name
  • Nomenclature group
  • Quantitatively:
    • Opening balance
    • Increase
    • Decrease
    • Closing balance
  • Summarily:
    • Opening balance
    • Increase
    • Decrease
    • Closing balance
Raw data file

Raw data records exist at the date the changes occur. We need to expand them to cover days without changes. This is required to correctly calculate the average balance

DemoData.pbix

You can explore data on this chart by selecting Date, month, Group or single Item, and all KPI's will updates. In this case Inventory Turnover rates like:

  • 0..5 days - High
  • 5..20 days - Normal
  • 20..50 days - Low
  • 50.. days - Critical

Analysis

The right-side chart represents average inventory cost by groups. Each group is highlighted by specific color, related to Inventory Turnover rate. So, we see two big groups with "Low" preformance.
